Some words you should know to run your server

Feb 18, 2016

There are some words you should know to learn more effectively. Don’t try to learn by heart at the same time. You will see more in this website. I will explain with a simple way to make it easy to understand. For full detail please search on Google or leave a comment below.

  • VPS (Virtual private server) A virtual private server is a virtual machine (not real). There are many virtual machines that can be run on one physical server.

  • Dedicated Server A dedicated server is exactly what it sounds like. The entire physical server is yours to use, and yours alone.

  • Hosting Hosting (Web Hosting) is the service providing space on the Internet for websites. To make your web site visible to the world, you’ll have to store it on a web server.The scope of web hosting services varies greatly. The most basic is web page and small-scale file hosting, where files can be uploaded via File Transfer Protocol (FTP) or a Web interface.

  • Website Domain: A website domain, often called a domain, is the name of the website or URL. You can see my domain is “vpsmaster.net”

  • DNS (Domain Name System)
    DNS is an abbreviation for Domain Name System, a system for naming computers and network services that is organized into a hierarchy of domains. DNS naming is used in TCP/IP networks, such as the Internet, to locate computers and services through user-friendly names.

  • IP address A unique string of numbers separated by periods. Example you type “ip” on google search, you can see your public IP in the first result.

  • CDN (Content Delivery Network) A content delivery network (CDN) is a system of distributed servers (network) that deliver webpages and other Web content to a user based on the geographic locations of the user, the origin of the webpage and a content delivery server. Using CDN will make your website faster.

  • Ping The ping is the reaction time of your connection.

  • Bandwidth Bandwidth is also defined as the amount of data that can be transmitted in a fixed amount of time.

  • Port Speed Port speed is in reference to the maximum amount of data that can pass through the port at a given second on the switches that connect your server to the Internet. The unit of measurement is in megabits per second (Mbps) or gigabits per second (Gbps).
